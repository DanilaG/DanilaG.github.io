What is a problem?

If you travel in a poorly mapping area (forest, desert, tundra, etc), it would be hard to navigate. All that you want to know at this moment: direction and distance to aim. In some cases, you would like to know how you got here and how you can come back the same way.

How to solve it?

By the apps. In perfectly you set a set of halt's locations from the smartphone and see direction and distance to them from a smartwatch. Also, you can save a start treap location. In this case, applications will be show distance and direction to it and a traversed path according to the current user’s location/rotation.

Who are your clients?

People that are work/travel in poorly mapping areas.

What are main features?

  • Connection watch and phone applications;
  • Work in magnetic fields (without the compass);
  • Show direction errors;
  • Combine tracker and “smartcompass” functionality.

How is scale down the project?

Implement next features:

  • save current location;
  • add locations from coordinates;
  • show direction and distance to them.

How is scale up the project?

Add:

  • share traversed paths;
  • create lib with itineraries, where users can add itineraries and comet them.

What does mean "Dersu"?

It is the first name of the main forest guide in Russian expeditions in Far East region. His full name is Dersu Uzala.
